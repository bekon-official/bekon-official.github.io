hexo与github连接时出现Could not read from remote repository

##博主的系统和版本：
win10
node.js
git.2.28.0

在跟据原文：基于Hexo+github+coding搭建个人博客
github pages + Hexo + 域名绑定搭建个人博客
等文章一步步指示下
‘数据已经成功备份到github 但在hexo deploy这一步上’

始终报错

参考此处https://blog.csdn.net/huanhuaqian/article/details/81986064

返回

好像没有问题，再检查SSH是否正常连接
cd ~/.ssh

出现 You’ve successfully authenticated, but GitHub does not provide shell access.说明显示正常

回去重新看了自己的yml配置

在网上找了114514种方法都不适用，不过可以确定问题出在在几个地址指向上
最后发现是GitHub取名的时候，**必须使用git@github.com:（用户名）/（用户名）.github.io.git为该格式，否则无法使用page的功能**
否则会造成能够hexo d 能上传，但是无法展示
